# Getting Started

This is a recipe site that you can use for searching recipes ! ! 




### but first, you need to do something !!

## Add the .env file 

```bash
VITE_APP_ID = ?;
VITE_APP_KEY = ?;
```

## Add the dependencies

```bash
npm install
# or
yarn
```

## Run the app

```bash
Npm run dev
#or
yarn dev
```
